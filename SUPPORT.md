# Support

## Getting help

- **Usage questions** ("how do I configure X?"): open a
  [GitHub Discussion](https://github.com/kero168/agentsync/discussions) if
  enabled, or open an issue using the "Question" template.
- **Bug reports**: open an issue using the "Bug report" template — see
  [CONTRIBUTING.md](./CONTRIBUTING.md).
- **Feature requests**: open an issue using the "Feature request" template.
- **Security issues**: do **not** open a public issue — see
  [SECURITY.md](./SECURITY.md).

## Before you ask

1. Check [docs/quickstart.md](./docs/quickstart.md) and the
   [README](./README.md) command reference.
2. Look at [examples/](./examples) for a working configuration close to
   your use case.
3. Run `agentsync diff` — it usually explains exactly what's out of sync
   and why.
4. Search existing issues for your error message.

## Response times

agentsync is maintained on a best-effort, volunteer basis (see
[GOVERNANCE.md](./GOVERNANCE.md)). There is no guaranteed SLA. Well-scoped
bug reports with a minimal reproduction are the fastest to act on.
